# Leul Abiti — Professional Portfolio

A high-performance, premium editorial-style portfolio showcasing the work of Leul Abiti, a Full-Stack Developer and Hardware Engineer based in Addis Ababa, Ethiopia.

## 🚀 Overview

This portfolio is designed to demonstrate expertise in delivering MVPs, scaling brands, and bridging the gap between software and hardware. It features a sleek, responsive design with interactive project previews and a dedicated focus on performance and typography.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern features like Custom Properties, Bento Grid, and Backdrop Filter), Vanilla JavaScript.
- **Interactions:** Intersection Observer API for reveal animations, custom 3D tilt effects, and a responsive live preview modal engine.
- **Design Aesthetic:** Premium Editorial (Light/Dark mode support).

## ✨ Key Features

- **Responsive Bento Grid:** A modern layout for showcasing selected works.
- **Interactive Live Previews:** View projects directly within the portfolio using a built-in device simulator (Desktop/Mobile).
- **Dark/Light Mode:** Seamless theme switching with system preference memory.
- **Performance Optimized:** Built with vanilla technologies to ensure fast load times and high Core Web Vitals scores.
- **Hardware-Software Focus:** Highlighting unique expertise in embedded systems and firmware development.

## 📂 Project Structure

- `index.html` — Main structure and content.
- `styles.css` — Modern, modular styling with CSS variables.
- `script.js` — Custom interaction logic and modal engine.
- `IMG_0033.webp` — Professional portrait.

## 💻 Local Development

To run this project locally:

1. Clone the repository.
2. Open `index.html` in your favorite browser.
3. (Optional) Use a local server like "Live Server" in VS Code for hot-reloading.
